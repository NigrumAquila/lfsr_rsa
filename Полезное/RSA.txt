p, q - случайные простые числа
n - произведение p на q, также называют модулем
fi - функция Эйлера, будет нужна для ограничения максимального значения открытой экспоненты e и для нахождения закрытой экспоненты d
e - открытая экспонента, это случайное простое число, взаимно простое со значением функции Эйлера от p,q
d - закрытая экспонента мультипликативная инверсия числа e по модулю fi
publicKey - это 2 числа, 1 является открытая экспонента, 2 - модуль
privateKey - это 2 числа, 1 является закрытая экспонента, 2 - модуль

шифрование происходит посредством возведения аргумента в степень открытой экспоненты по модулю
дешифрование происходит посредством возведения шифротекста в степень закрытой экспоненты по модулю

мультипликативная инверсия - это обратное число b к числу a,
уравнение: a*b mod n == 1,
для нахождения числа b используют расширенный алгоритм Евклида, подробнее сам алгоритм можно рассмотреть на этой странице https://en.wikipedia.org/wiki/Extended_Euclidean_algorithm
однако, пример на этой странице не будет полезным для криптографии, т.к. мы ищем мультипликативную инверсию, а для этого, нужно чтобы числа a, n были взаимно простыми, то есть имели НОД==1 
рабочий пример: a = 6, n = 19, тогда число b = 16
проверка: 6*16 mod 19 = 96 mod 19 = 1
